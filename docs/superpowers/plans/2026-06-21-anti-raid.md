# Anti-Raid Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add automatic bans for very recent Discord accounts and mention spam.

**Architecture:** Keep Discord event handling in a global JDA listener and place time-window decisions in a small service. The service is covered by unit tests; the listener stays thin and uses the existing auto-discovery, env configuration, ban, and log-channel patterns.

**Tech Stack:** Java 25, JDA 6.4.1, Gradle, JUnit Jupiter, SLF4J/Logback.
